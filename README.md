## 🎯 Face Recognition Attendance System
A simple Face Recognition Attendance System built using Python and OpenCV. 
The system detects and recognizes faces through a webcam and automatically marks attendance with date and time.

## ✨ Features

- Real-time face detection
  
- Face recognition using LBPH algorithm
  
- Automatic attendance marking
  
- Attendance stored in CSV file
  
- Easy to use and lightweight

## 🛠️ Technologies Used

- Python
  
- OpenCV
  
- NumPy

## 📂 Project Structure

project/
│
├── images/
├── Attendance.csv
├── main.py
└── README.md

## 🚀 How to Run

1. Install required libraries: pip install opencv-contrib-python numpy

2. Add face images to the "images" folder.

3. Run the program: python main.py

4. The webcam will open and attendance will be recorded automatically when a registered face is detected.

## 📊 Output

Attendance is saved in:

Attendance.csv

with:
- Name
  
- Time
  
- Date
